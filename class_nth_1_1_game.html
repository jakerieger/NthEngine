Detailed Description

Base class where all engine systems meet and execute.

Central hub for the game engine that owns and manages all core systems including the window, rendering, input, audio, scripting, and scene management. Handles the main game loop and communication between the OS, graphics API, and game client code. User games should inherit from this class and override lifecycle methods to implement custom behavior.
