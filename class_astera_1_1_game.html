Detailed Description

Base class where all engine systems meet and execute.

Extends Window to add game-specific functionality including rendering, audio, scripting, and scene management. Handles the main game loop and communication between the OS, graphics API, and game client code. User games should inherit from this class and override lifecycle methods to implement custom behavior.
