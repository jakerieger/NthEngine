Detailed Description

High-precision clock for timing and frame rate tracking.

Provides accurate time measurements for delta time calculation, frame rate monitoring, and total elapsed time tracking. Uses platform-specific high-resolution timers (QueryPerformanceCounter on Windows, clock_gettime on other platforms).
