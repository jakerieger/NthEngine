Detailed Description

Manages keyboard and mouse input state for the game.

Provides polling-based input handling for keyboard keys and mouse buttons, as well as mouse position and delta tracking. Input state is updated internally by the Game class and can be queried at any time. Also supports Lua scripting integration.
